# Arduino Weather Station

A weather station built with Arduino UNO that measures enviromental conditions using multiple sensors.

## Features

- Temperature (LM35)
- Temperature & Humidity (DHT11)
- Atmospheric Pressure (BMP280)
- Altitude Estimation (BMP280)
- Ambient Light Detection (LDR)
- Air Quality Monitoring (MQ135)
- LCD 20x4 Display (Parallel Interface)
- LCD 20x4 I2C Display
- Multi-page menu using push button

## Project Development

This repository contains the complete development process of the weather station.

Instead of uploading only the final code, every stage of the project is included, from the first prototype to the final version.

### Development Steps

- Step 1 – LCD + LM35
- Step 2 – LCD I2C (upgrade) + DHT11 + Push Button
- Step 3 – BMP280 + LDR
- Step 4 – MQ135
- Step 5 – Final Weather Station


## Hardware

- BMP280
